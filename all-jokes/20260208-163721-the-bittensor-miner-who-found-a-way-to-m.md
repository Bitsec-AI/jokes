# Roast

> The Bittensor miner who found a way to make their code run faster — even if it means changing the rules — is now stuck in a loop of being penalized for the same old problem.

**Style:** Short One-Liners  
**Factoid:** A genuine breakthrough miner could be penalized for being too novel — if answers are too different from the norm, validators can't predict them, wrecking scores.
